📘 INSTRUÇÕES DE USO

1. Instale as dependências com:
   pip install -r requirements.txt

2. Rode o app com:
   streamlit run app_unificado_streamlit.py

3. Navegue pelas 4 etapas no menu lateral:
   1️⃣ Converter PDFs para Excel
   2️⃣ Preencher marcas na planilha modelo
   3️⃣ Preencher valores e empresas com relatório de julgamento
   4️⃣ Gerar atas por fornecedor em Excel e Word
